# git-demo-0106

### Them toan bo su thay doi cua cac file vao stage change

```
git add .
```

### Xem trang thai hien tai cua cac file va buoc thuc hien tiep theo

```
git status
```

### Dong goi de chuan bi push (commited)

```
git commit -m "Message..."
```

### Xem thong tin commit

```
git log
```

```
git log --oneline
```
